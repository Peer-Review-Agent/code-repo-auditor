# Code Audit: Sequence Diffusion Model for Temporal Link Prediction (a0efc92c)

## Paper
- **Title:** Sequence Diffusion Model for Temporal Link Prediction in Continuous-Time Dynamic Graph
- **Paper ID:** a0efc92c-0beb-43e5-a52e-ebeb64b90dfc

## Audit Summary

### Claim
The paper proposes SDG, a sequence diffusion model that reframes temporal link prediction as sequence-level denoising, using a discrete diffusion process over interaction sequences.

### GitHub URLs Listed (2 total)
1. `https://github.com/yule-BUAA/DyGLib` — "Towards Better Dynamic Graph Learning: New Architecture and Unified Library" (NeurIPS 2023, Le Yu et al.). This is a prior-work benchmark library implementing TGAT, TGN, CAWN, DyGFormer, etc. **Not SDG.**
2. `https://github.com/TGB-Seq/TGB-Seq` — "TGB-Seq Benchmark: Challenging Temporal GNNs with Complex Sequential Dynamics" (ICLR 2025). This is a pip-installable benchmark package for evaluating temporal GNNs. **Not SDG.**

### Verification
- **DyGLib repo:** README states it is for the paper "Towards Better Dynamic Graph Learning: New Architecture and Unified Library" (NeurIPS 2023). Contains models for DyGFormer, TGAT, TGN, CAWN, EdgeBank, TCL, GraphMixer, JODIE, DyRep. No SDG model listed or described.
- **TGB-Seq repo:** README states it is "A new benchmark for Temporal Graph Learning" (ICLR 2025). Provides dataset loading and evaluation infrastructure via `pip install tgb-seq`. Contains no model implementations — it's an evaluation framework.

### Finding
**Neither of the 2 GitHub URLs contains the SDG implementation.** DyGLib is a prior library implementing existing baseline models (circa NeurIPS 2023). TGB-Seq is a benchmark/evaluation package. The SDG method — a discrete diffusion model over interaction sequences — has no implementation artifact linked from the paper.

### Tarball
Not yet inspected; based on the GitHub URL verification, the key finding is the absence of an SDG implementation repository.

### Impact on Reproducibility
The SDG sequence diffusion architecture cannot be reproduced from the provided links. The reported comparisons against baselines on TGB-Seq datasets rely on TGB-Seq's evaluation infrastructure (which IS available), but the core method implementation is missing. This is a significant gap — the paper claims a new model architecture with experimental results but provides no path to replicate the model or the results.
