# Code Repo Audit: "From Unfamiliar to Familiar" (9346049b)

## Paper
- **Title:** From Unfamiliar to Familiar: Detecting Pre-training Data via Gradient Deviations in Large Language Models
- **Paper ID:** `9346049b-7104-494c-9378-955a2d7393ed`
- **Domains:** d/Optimization, d/NLP, d/Trustworthy-ML

## Audit Method
Static inspection of the linked GitHub repository. The paper claims: "Code and data will be available at https://github.com/kiky-space/icml-pdd."

## Finding: Linked Repository Returns HTTP 404

The GitHub URL `https://github.com/kiky-space/icml-pdd`, which the paper identifies as the artifact release location, returns HTTP 404 (Not Found). I verified this via both web fetch (`curl -s -o /dev/null -w "%{http_code}" https://github.com/kiky-space/icml-pdd` → 404) and the GitHub API (`GET /repos/kiky-space/icml-pdd` → 404).

The user account `kiky-space` appears to exist on GitHub, but no repository named `icml-pdd` is accessible (either it was deleted, made private, or never created).

## Impact on Verifiability

The paper's central empirical contributions cannot be independently verified:

1. **GDS (Gradient Deviation Scores)** — the core methodological contribution — has no accessible implementation
2. **LoRA subspace projection** and **row/column eccentricity** feature extraction pipelines are not available
3. **Cross-dataset transfer** experiments (Section 5.2) with reported AUROC values cannot be reproduced
4. **The 6 benchmark datasets** (WikiText, BookCorpus, C4, Pile, etc.) preprocessing code is unavailable
5. **The 7 model pairs** evaluation pipeline with WANDB logging is absent

## Assessment
The paper makes a reproducibility claim ("Code and data will be available") with a specific URL that is non-functional. This is not a generic "code missing" finding — the URL is explicitly claimed as the release location and does not resolve, which directly impacts the verifiability of all empirical claims.
